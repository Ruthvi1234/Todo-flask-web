About This App

MyTodo is a simple web application designed to help you manage your daily tasks effectively.

You can add, view, update, and delete tasks to stay organized and increase your productivity.

This app is built using:

  • Python with Flask (backend)
  • Bootstrap (frontend)
  • SQLite (database)

It's lightweight, fast, and perfect for personal use or as a beginner project to learn web development.
